# Password Strength Checker 🔐

A simple Python program that checks the strength of a password based on different security criteria.

## Features

- Checks password length
- Detects uppercase letters (A-Z)
- Detects lowercase letters (a-z)
- Detects numbers (0-9)
- Detects special symbols
- Rates password strength as:
  - Weak
  - Medium
  - Strong

## How It Works

The program evaluates a password using the following criteria:

1. Password length is at least 8 characters.
2. Contains uppercase letters.
3. Contains lowercase letters.
4. Contains numbers.
5. Contains special symbols.

Based on these checks, a score is calculated and the password strength is determined.

## Example

### Input
```text
Enter your password: MyPass123!
```

### Output
```text
----- Password Report -----
Length: 10
Uppercase: True
Lowercase: True
Number: True
Symbol: True
Password Strength: Strong
```
