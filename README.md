# 🔐 Project-1-Password-Strength-Checker---DecodeLabs-Cyber-Security-Internship

A Python script that checks password strength based on length and character variety (uppercase, lowercase, numbers, and special characters) — Project 1 of my Cyber Security Internship at Decode Labs.

## Features
- Password length validation
- Uppercase letter detection
- Lowercase letter detection
- Number detection
- Special character detection
- Password strength classification (Weak, Medium, Strong)
